# Mini Patent AI

## Overview
Mini Patent AI is a project focused on leveraging artificial intelligence to assist with patent applications, reviews, and searches. The system is designed to streamline the patent process, making it more efficient and accessible.

## Features
- **AI-Powered Search**: Quickly find relevant patents using advanced AI algorithms.
- **Application Assistance**: Offers guidance throughout the patent application process.
- **Review Tools**: Provides tools for reviewing patent documents with ease.
- **User-Friendly Interface**: Easy to navigate interface catering to both novice and expert users.

## Installation
To set up this project on your local machine, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/octaboomai/mini-patent-ai.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mini-patent-ai
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```

Visit `http://localhost:3000` in your browser to access the Mini Patent AI interface.
